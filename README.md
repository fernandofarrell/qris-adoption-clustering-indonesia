# Assessing QRIS Adoption Efficiency In Indonesia Through Clustering Analysis

This repository contains the dataset and computational code used in the research paper: **"Assessing QRIS Adoption Efficiency In Indonesia Through Clustering Analysis"** (2024).

## Files in this Repository:
* `qris_regional_dataset_2024.csv`: The pre-processed dataset containing the IP-TIK Score, standardized User Ratio, and standardized Merchant Ratio for 34 provinces in Indonesia. Data sources include the Central Bureau of Statistics (BPS) and Bank Indonesia.
* `kmedoids_clustering_analysis.ipynb`: The Python implementation of the K-Medoids algorithm, Elbow Method evaluation, and statistical validation (ANOVA & Tukey HSD) used in this study.
